# 🃏 Cards Project

## 📌 Overview
This is a modern and responsive card-based project built using HTML, CSS, JavaScript, and Bootstrap.  
The project focuses on creating interactive UI cards with a clean design and dynamic behavior using JavaScript.

---

## ✨ Features
- Responsive card layout using Bootstrap  
- Clean and modern UI design  
- Interactive cards using JavaScript  
- Dynamic content handling  
- Mobile-friendly design  

---

## 🛠️ Technologies Used
- HTML5  
- CSS3  
- JavaScript (ES6)  
- Bootstrap 5  

---

## 🎯 Purpose
This project was built to practice DOM manipulation, Bootstrap layout system, and building interactive UI components using JavaScript.
